Course Learning Reflections

What are the kinds of problems we see in nature?

Problems in nature can be categorized into different types such as:

  • Iteration: Repeating a process multiple times, for example, iterating over an array of crops for irrigation planning.
  • Recursion: A problem that calls itself repeatedly, such as breaking down the larger issue of crop management into smaller recursive tasks (e.g., dividing fields into sub-sections).
  • Backtracking: This approach is useful for exploring different possibilities, like trying different irrigation methods and backing out if a particular approach fails to optimize water usage.

What is space and time efficiency? Why are they important?

Space and time efficiency are crucial in designing algorithms that handle large amounts of data efficiently. Time efficiency refers to the speed at which an algorithm executes, while space efficiency refers to the amount of memory it consumes. For example, when optimizing irrigation systems, we want the algorithm to run quickly and consume as little memory as possible.

Take away from design principles (Chapter 2)

Chapter 2 emphasizes the importance of efficient problem-solving strategies. Principles like divide and conquer, dynamic programming, and greedy algorithms are essential. For example, using Dijkstra’s Algorithm for distributing crops optimally or applying dynamic programming for water usage planning.

How hierarchical data and tree data structures optimize problems?

Hierarchical data structures like trees (e.g., Binary Search Tree, AVL) are critical for organizing and managing data in agricultural systems. For instance, a farm’s crop yield data could be stored in a tree structure to optimize search times and ensure faster data retrieval for planning.

The need for array query algorithms

Array query algorithms, such as segment trees or Fenwick trees, are necessary for efficiently answering range queries, such as calculating total water consumption across a set of fields or determining crop yield within a given range of dates.

Difference between trees and graphs

Trees are hierarchical structures, ideal for applications like hierarchical crop data storage, while graphs are more flexible and are suitable for representing the transportation network in agricultural areas.

Sorting and Searching Algorithms

Sorting algorithms like Merge Sort or Quick Sort are vital for organizing crop data for better management. Searching algorithms, such as binary search, help in quickly finding relevant information, like identifying crops that need more water.

Importance of Graph Algorithms

Graph algorithms like Dijkstra’s and Prim’s algorithm are essential for optimizing transportation paths, ensuring crops are efficiently distributed, and minimizing transportation costs across fields.

Different Algorithm Design Techniques

Techniques such as divide and conquer, dynamic programming, and greedy algorithms offer distinct benefits for problem-solving. For example, divide and conquer can break down complex issues like crop management into smaller, manageable parts for more efficient solutions.

Justification:

Mixing fields of different crops: Tools: Graph (BFS/DFS), Dijkstra's Algorithm These tools help model fields as a network, identifying the shortest or most efficient paths between fields for organizing crop placement and harvest logistics. Graph traversal methods (BFS/DFS) allow exploring connectivity, while Dijkstra’s algorithm optimizes the movement of resources between fields.

Excessive wastage of water during irrigation: Tools: Segment Trees, Fenwick Trees These data structures are efficient for managing and querying range-based data, such as the distribution of water to different fields. They allow quick updates and queries for water levels, ensuring even distribution across multiple fields.

No proper disposal of weeds: Tools: Hashing, Traveling Salesman Problem (TSP), Prim's Algorithm Hashing efficiently tracks weed disposal locations to avoid revisiting them. TSP models optimal routes for weed disposal teams, minimizing effort and time, while Prim’s algorithm aids in creating minimum-cost paths for managing weed transport.

Unorganized distribution of harvested products: Tools: Heap, Dijkstra's Algorithm Heap structures prioritize distribution tasks (e.g., areas with higher demand), while Dijkstra’s algorithm optimizes the delivery paths, ensuring balanced and efficient product distribution to urban markets.

Excessive use of pesticides and insecticides: Tools: Sparse Tables Sparse Tables efficiently manage range queries for chemical usage over time, enabling the identification of overused areas. They allow fast computation without additional memory overhead, helping maintain balance and quality in pesticide application.

Transportation in agricultural areas: Tools: Kruskal’s, Prim’s Algorithm Both algorithms are suitable for designing minimal-cost networks. Kruskal’s algorithm finds the most efficient way to connect all fields with minimal resources, while Prim’s algorithm builds optimized, continuous routes for transportation within the agricultural area.

void checkThresholdAlert(const vector& pesticide_usage, int threshold) {
    for (int i = 0; i < pesticide_usage.size(); ++i) {
        if (pesticide_usage[i] > threshold) {
            cout << "Alert: Excessive pesticide usage in region " << (i + 1) << ": " << pesticide_usage[i] << endl;

}

double calculateAverage(const vector& pesticide_usage) {
    int sum = 0;
    for (int i = 0; i < pesticide_usage.size(); ++i) {
        sum += pesticide_usage[i];
    }
    return static_cast(sum) / pesticide_usage.size();
}

double calculateStdDev(const vector& pesticide_usage, double avg) {
    double variance = 0;
    for (int i = 0; i < pesticide_usage.size(); ++i) {
        variance += pow(pesticide_usage[i] - avg, 2);
    }
    return sqrt(variance / pesticide_usage.size());
}

string detectTrend(const vector& pesticide_usage) {
    int count_increase = 0;
    int count_decrease = 0;

    for (size_t i = 1; i < pesticide_usage.size(); ++i) {
        if (pesticide_usage[i] > pesticide_usage[i - 1]) {
            count_increase++;
        } else if (pesticide_usage[i] < pesticide_usage[i - 1]) {
            count_decrease++;
        }
    }

    if (count_increase > count_decrease) {
        return "Increasing usage trend detected";
    } else if (count_decrease > count_increase) {
        return "Decreasing usage trend detected";
    } else {
        return "No significant trend detected";
    }
}

void buildSparseTable(const vector& arr, vector >& st, vector& log) {
    int n = arr.size();
    log[1] = 0;
    for (int i = 2; i <= n; ++i) {
        log[i] = log[i / 2] + 1;
    }

    for (int i = 0; i < n; ++i) {
        st[i][0] = arr[i];
    }

    for (int j = 1; (1 << j) <= n; ++j) {
        for (int i = 0; i + (1 << j) - 1 < n; ++i) {
            st[i][j] = min(st[i][j - 1], st[i + (1 << (j - 1))][j - 1]);
        }
    }
}

int query(int L, int R, const vector >& st, const vector& log) {
    int length = R - L + 1;
    int j = log[length];
    return min(st[L][j], st[R - (1 << j) + 1][j]);
}

void findClusters(int n, string fields[], double distances[MAX_FIELDS][MAX_FIELDS], bool visited[], int clusters[MAX_FIELDS]) {
    int clusterId = 0;
    for (int i = 0; i < n; i++) {
        if (!visited[i]) {
            clusters[i] = clusterId;
            visited[i] = true;

            for (int j = 0; j < n; j++) {
                if (distances[i][j] > 0 && !visited[j]) {
                    visited[j] = true;
                    clusters[j] = clusterId;
                }
            }

            clusterId++;
        }
    }
}

void dijkstra(int n, double distances[MAX_FIELDS][MAX_FIELDS], int start, string fields[]) {
    double dist[MAX_FIELDS];
    bool visited[MAX_FIELDS] = {false};

    for (int i = 0; i < n; i++) {
        dist[i] = numeric_limits::infinity();
    }
    dist[start] = 0;

    for (int i = 0; i < n; i++) {
        int u = -1;
        double minDist = numeric_limits::infinity();

        for (int j = 0; j < n; j++) {
            if (!visited[j] && dist[j] < minDist) {
                minDist = dist[j];
                u = j;
            }
        }

        if (u == -1) break;

        visited[u] = true;

        for (int v = 0; v < n; v++) {
            if (distances[u][v] > 0) {
                double newDist = dist[u] + distances[u][v];
                if (newDist < dist[v]) {
                    dist[v] = newDist;
                }
            }
        }
    }

    cout << "\nShortest paths from " << fields[start] << ":" << endl;
    for (int i = 0; i < n; i++) {
        if (dist[i] == numeric_limits::infinity()) {
            cout << "  To " << fields[i] << ": No direct path." << endl;
        } else {
            cout << "  To " << fields[i] << ": " << fixed << setprecision(2) << dist[i] << " units" << endl;
        }
    }
    cout << endl;
}

void buildTree(vector& arr, int start, int end, int node) {
    if (start == end) {
        tree[node] = arr[start];
    } else {
        int mid = (start + end) / 2;
        buildTree(arr, start, mid, 2 * node + 1);
        buildTree(arr, mid + 1, end, 2 * node + 2);
        tree[node] = tree[2 * node + 1] + tree[2 * node + 2];
    }
}

void updateRangeUtil(int start, int end, int l, int r, int value, int node) {
    if (lazy[node] != 0) {
        tree[node] += (end - start + 1) * lazy[node];
        if (start != end) {
            lazy[2 * node + 1] += lazy[node];
            lazy[2 * node + 2] += lazy[node];
        }
        lazy[node] = 0;
    }
    if (start > end || start > r || end < l) {
        return;
    }
    if (start >= l && end <= r) {
        tree[node] += (end - start + 1) * value;
        if (start != end) {
            lazy[2 * node + 1] += value;
            lazy[2 * node + 2] += value;
        }
        return;
    }
    int mid = (start + end) / 2;
    updateRangeUtil(start, mid, l, r, value, 2 * node + 1);
    updateRangeUtil(mid + 1, end, l, r, value, 2 * node + 2);
    tree[node] = tree[2 * node + 1] + tree[2 * node + 2];
}

void updateRange(int l, int r, int value) {
    updateRangeUtil(0, lazy.size() / 4 - 1, l, r, value, 0);
}

int queryRangeUtil(int start, int end, int l, int r, int node) {
    if (start > end || start > r || end < l) {
        return 0;
    }
    if (lazy[node] != 0) {
        tree[node] += (end - start + 1) * lazy[node];
        if (start != end) {
            lazy[2 * node + 1] += lazy[node];
            lazy[2 * node + 2] += lazy[node];
        }
        lazy[node] = 0;
    }
    if (start >= l && end <= r) {
        return tree[node];
    }
    int mid = (start + end) / 2;
    int left = queryRangeUtil(start, mid, l, r, 2 * node + 1);
    int right = queryRangeUtil(mid + 1, end, l, r, 2 * node + 2);
    return left + right;
}

int queryRange(int l, int r) {
    return queryRangeUtil(0, lazy.size() / 4 - 1, l, r, 0);
}

void simulateRegrowth(int location, int timeSinceClearance) {
    float regrowthProbability = (timeSinceClearance / 10.0f) * (static_cast(rand()) / RAND_MAX);

    if (regrowthProbability > REgrowth_THRESHOLD) {
        cout << "Regrowth detected at location " << location << ". Re-clearing required.\n";
        regrowthHistory[location]++;
    }
}

void detectPotentialRegrowth(int location, Environment env) {
    if (env.moisture > 50.0f && env.temperature > 20.0f) {
        cout << "Potential regrowth likely at location " << location
             << " due to moisture: " << env.moisture
             << "% and temperature: " << env.temperature << "°C\n";

        if (regrowthHistory.find(location) == regrowthHistory.end()) {
            regrowthHistory[location] = 0;
        }
        regrowthHistory[location]++;
    }
}

void dynamicRegrowthCheck(int location) {
    if (rand() % 100 < 30) {
        cout << "Regrowth detected at location " << location << " during traversal.\n";
        regrowthHistory[location]++;
    }
}

void simulateWeedClearing(vector route, int timeSinceClearance[], Environment env[]) {

        int location = route[i];
        cout << "Clearing weeds at location " << location << endl;


        detectPotentialRegrowth(location, env[location]);

        dynamicRegrowthCheck(location);

        simulateRegrowth(location, timeSinceClearance[location]);

        cout << endl;
    }
}
